# WhereIsWaldo
Projet final de Reconnaissance Visuelle implémentant un solveur du jeu « Où est Charlie ? ».

Contributeurs : François Petit et Fabien Maës.

Description du dépôt :
- le dossier data_bow contient l'ensemble des vignettes Charlie (versions rognées des planches centrées sur Charlie) ;
- le dossier original-images contient l'ensemble des planches ;
- le fichier pdf FP_maes+petit.pdf constitue le rapport du projet ;
- le Notebook detection_charlie.ipynb contient l'ensemble du code implémentant la chaîne de traitements décrite dans le rapport.
